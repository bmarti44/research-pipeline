# Claude Code Preferences

## Coding Style Preferences

### Type Hints
- **Always use type hints in code** - Include type annotations for function parameters, return types, and variables where appropriate.

### Variable Naming
- **Use descriptive variable names** - Prefer clear, meaningful variable names instead of short or abbreviated ones. Code readability is prioritized over brevity.
